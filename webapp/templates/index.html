Hello!

This is a passion project relating to SSENSE (https://www.ssense.com/en-ca), one of my favorite e-commerce retailers.

Together with the plan to scrape data from the web and create my own dataset for a machine learning project, I decided to try scraping product data from SSENSE by utilizing the AWS Lambda Function. Then, I built a recommendation engine based on analyzed and processed text details of the products.
